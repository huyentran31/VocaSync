🛡 Grounding Gate (Python, 0 AI)

“The AI proposes, the transcript decides.”

• every quoted line must exist verbatim in the transcript
• fabricated quotes replaced with the real cue
• timestamps come from subtitle cues, never from the model
• ungrounded rows flagged ⚠ for review

📚 Local Knowledge

WordNet — offline senses & relations (primary source). ConceptNet — optional online lookup. Otherwise data stays local; the LLM API is the only other network call.

↳

✅ Human-in-the-Loop Review

The learner aligns with the AI: review every row, edit any field if needed, and approve. Editable table — ⚠ ai-flag (why a row needs eyes) · 🤖 ai-fields (what the AI wrote) · sense browser · ungrounded rows carry the nearest real transcript line. Nothing enters the graph until you commit.
